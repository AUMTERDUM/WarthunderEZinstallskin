# Deploy to Railway.app (Free Tier)

## ขั้นตอน:

1. สมัคร Railway: https://railway.app
2. เชื่อม GitHub account
3. New Project → Deploy from GitHub repo
4. เลือก `AUMTERDUM/WarthunderEZinstallskin`
5. Railway จะ detect Node.js และ deploy อัตโนมัติ

## Environment Variables (ถ้าต้องการ)
- `PORT`: 3000 (Railway จะตั้งให้อัตโนมัติ)

## หมายเหตุ:
- Railway Free Tier: $5 credit/month (~500 hours)
- จะได้ domain เช่น: `your-app.railway.app`
- รองรับ file upload และ Node.js

## ทางเลือกอื่น:
- **Render.com**: Free tier (จำกัด 750 hours/month)
- **Fly.io**: Free tier พร้อม persistent disk
